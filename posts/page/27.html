Javascript의 Date.getTimezoneOffset() 오동작

Windows 의 제어판에서 GMT를 변경했는데 Javascript의 Date.getTimezoneOffset() 리턴값이 변경되지 않는다면 Google Chrome, Firefox의 경우 리스타트를 해줘야 한다. Internet Explorer는 상관없다. <script type=“text/javascript”>     var curDateTime = new Date();     var gmtOffsetHour = Math.floor(-(curDateTime.getTimezoneOffset()/60));     if(curDateTime.getTimezoneOffset() > 0) {         gmtOffsetHour = Math.ceil(-(curDateTime.getTimezoneOffset()/60));     }     var gmtOffsetMin = (-(curDateTime.getTimezoneOffset() % 60));     gmtOffsetMin = Math.abs(gmtOffsetMin);         var gmtOffst = gmtOffsetHour + “:” + ((gmtOffsetMin == 0) ? “0” : “”) + gmtOffsetMin;     alert(‘gmtOffst=’ + gmtOffst); </script> --
