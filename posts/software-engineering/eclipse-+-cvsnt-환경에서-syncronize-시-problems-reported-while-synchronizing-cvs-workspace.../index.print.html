eclipse + cvsnt 환경에서 "Syncronize" 시 "Problems reported while synchronizing CVS Workspace..."

- cvsNT 2.0.4

- Eclipse 3.1.0

에서 발생함.

cvsNT의 문제임

repository path 에 드라이브문자가 들어가면 안됨

cvsNT 설청창에서 ‘repository prefix’를 사용할것

예를들어 repository가 ‘/cvs/repository’ 이런식으로 되도록 함
